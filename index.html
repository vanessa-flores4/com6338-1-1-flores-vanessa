Instructions

In this assignment, you will create a simple function using JavaScript that will make some small talk with the user using alert, confirm, and prompt when the user clicks the "Greetings!" button above.

First, create a function named greet.

function greet() {}

The button at the top of this page is already coded to run the greet function when clicked, so there is no need to modify the HTML. Here is the included code for the aforementioned button:

<button onclick="greet()">Greetings!</button>

After creating the greet function, inside the function, do the following:

  1. Use a prompt to ask the user for her name.
    • The user should be prompted with the text "What is your name?"
  2. Use alert to greet the user with her name.
    • E.g. if the user entered "Sally", the alert should display "Hello, Sally"
  3. Use another prompt call to ask the user their age.
    • The user should be shown the text "How old are you?"
    • Use the parseInt method to convert the user's answer into a number.
  4. Use a confirm to ask the user if she has had a birthday yet this year.
  5. Use an alert to tell the user what year she was born in.
    • If the user has had a birthday this year, subtract their age from the current year.
    • If the user has not yet had a birthday this year, subtract their age plus one from the current year.
    • Use new Date().getFullYear() to get the current year.
